# FlatManager

This app will be managing co-living with flatmates. 
First functionality which this app will have will be tracking whos turn is to clean up.

## Features:
  * Login screen using google login.
  * Option to create new flat or join existing flat
  * As a creator of new flat I can share invitation code so other flatmates can join 
  * Screen when I can clearly see "clener" for this week, status and whos turn is next week. On this screen the user who has cleaning duty has cleaning confirmation button.
  * Flats and users data will be hold at firebase database
  * Possible notification which will be notifing upcoming or unfinished duty.
  * Logout option
